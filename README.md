## Cui 4 In 1 Img To Url

## 名称：四合一图床上传程序

## 版本：1.0[如API未失效将不再更新维护]

## 特色：主要利用CUI制作的一款四合一图床上传程序，集合百度，搜狗，新浪，360图床为一体的一键化图床，并能一键生成html,ubb,markdown显示图片格式一键复制，仅约一款8kb单页就能实现功能。
